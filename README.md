記憶卡配對遊戲 (Memory Match)
純前端(HTML/CSS/JavaScript)的小型記憶卡配對遊戲: 翻開兩張卡片,符號相同就配對成功,在倒數時間內把所有卡片配對完成即獲勝。

功能
4 組配對(共 8 張卡片),使用水果符號(🍎🍌🍇🍓)。
60 秒倒數計時,時間歸零則結束並提示。
重新開始按鈕可重置牌面並重新開始倒數。
執行方式
方式 A: 直接開啟 (最簡單)
下載/複製專案檔案。
用瀏覽器直接開啟 index.html。
若瀏覽器因安全限制而阻擋 ES Modules,請改用方式 B。

方式 B: 用本機靜態伺服器
以任一靜態伺服器啟動即可,例如:

bash

複製代碼
# Python 3
python -m http.server 8000
然後開啟 http://localhost:8000/。

檔案結構
index.html: 頁面骨架(提供 board/timer/restart 的 DOM 容器)。
style.css: 版面與卡片樣式(使用 CSS Grid 排版)。
script.js: 遊戲邏輯(洗牌、翻牌、配對檢查、倒數計時、重新開始)。
遊戲規則
點擊卡片翻開符號。
一次最多翻開兩張;若符號相同則保持翻開並計入配對。
若符號不同,短暫延遲後自動蓋回。
在時間內完成全部配對會顯示勝利提示。
客製化
可在 script.js 中調整:

卡片內容: 修改符號陣列(例如換成動物或數字)。
遊戲時間: 修改預設倒數秒數。
牌面大小與排版: 在 CSS 中調整 grid 與卡片尺寸。
已知限制
目前勝利/失敗以 alert() 顯示提示。
基礎版本的洗牌使用 Array.sort(() => Math.random() - 0.5) 會有隨機分布不均問題;建議改為 Fisher–Yates 以提升洗牌品質。
